极客币

什么是极客币 & 如何获得？

极客币（JKB）是极客学院的虚拟货币，标识为 ，100 极客币相当于人民币 1 元钱。极客币可以通过 购买 VIP 会员，以及参加极客学院的各种活动等方式获得。

极客币可以用来做什么？

极客币可以用于在购买 职业学院 课程时抵扣商品价格，目前极客学院职业学院包括 Web 前端工程师, Python Web 工程师，Go语言工程师 和 iOS工程师 四个方向，购买这些职业方向的课程均可以使用极客币抵扣。

使用方法

  1. 选择喜欢的职业学院课程

  2. 点击「立即报名」按钮

  3. 在报名信息确认页面勾选「使用极客币抵扣 *.** 元」，之后进入支付流程即可

* 购买VIP会员不可使用极客币抵扣
